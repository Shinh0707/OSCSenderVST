## OSCSender
<!-- ↑は自動で入力されるので##のままにする -->
<!-- 説明文 (何をする目的のVSTか。ユーザは何を入力して、何を得られるのか。どういうことに使えるか。) -->
OSCSenderは、DAWのMIDIトラックに入力されたMIDIノート情報を、ネットワーク経由でOSC (Open Sound Control) メッセージとして送信するVST3/AUプラグインです。

DAW内で作成したMIDIシーケンスやリアルタイムのMIDI演奏を用いて、TouchDesigner、Resolume、Unity、Max/MSPなどのOSC対応ソフトウェアを制御したい場合に最適です。同一PC内での連携（ローカルホスト）はもちろん、LAN内の別コンピュータへもIPアドレスを指定して送信可能です。

### How to use
<!-- 使い方 -->
1. DAW（Ableton Live, Logic Pro, Studio One等）のMIDIトラックまたはインストゥルメントトラックに本プラグイン「OSCSender」をインサートします。
2. プラグインのUIを開き、以下の項目を設定します：
   - **IP Address**: 送信先のIPアドレス（同一PC内の別ソフトウェアに送る場合は `127.0.0.1` を指定）
   - **Port**: 送信先のポート番号（受信側ソフトウェアのOSC入力ポートに合わせます）
   - **OSC Address**: 送信するOSCメッセージのベースとなるアドレス（例: `/midi/note`）
3. トラックでMIDIクリップを再生、またはMIDIキーボードで入力を行うと、ノートオン/オフのタイミングでOSCメッセージが送信されます。

**送信されるOSCメッセージのフォーマット**
ベースアドレスを `/midi/note`、ノート番号を `60`（C3）として入力した場合、以下の2つのメッセージが同時に送信されます。

- ノートのオン / オフ状態（1 または 0 の `Int32`）: `/midi/note/note60` 
- ベロシティ値（0.0〜1.0 の `Float32`）: `/midi/note/note_vel60`

### Usage example
<!-- 使用例 -->
**映像ソフト（TouchDesigner / Resolume等）との同期**
DAW上でキック、スネア、ハイハットなどのリズムごとにMIDIノートを打ち込んだクリップを作成し、そのトラックにOSCSenderを挿入します。映像用ソフトウェア側でOSCを受信することで、楽曲のBPMやリズムに完全に同期した映像効果の切り替えや、パラメーターのモジュレーションを実現できます。

**複数PCを用いたパフォーマンスシステム**
音楽制作用のPCでDAWを動かし、映像描画専用の別PCへLAN経由でノート情報を送信することができます。これにより、1台のPCに負荷を集中させることなく、安定したオーディオビジュアルのパフォーマンス環境を構築できます。
